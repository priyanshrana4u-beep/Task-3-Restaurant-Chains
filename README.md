# Task-3-Restaurant-Chains

## 📌 Project Overview
This project analyzes restaurant chains using a restaurant dataset. The objective is to identify restaurant chains with multiple outlets and evaluate their performance based on average ratings and customer popularity (votes).

## 🎯 Objectives
- Identify restaurant chains in the dataset.
- Analyze the average rating of each restaurant chain.
- Analyze the popularity of restaurant chains using customer votes.
- Visualize the analysis using bar charts.

## 📂 Dataset
The dataset contains restaurant information, including:
- Restaurant Name
- Rating
- Votes
- Location
- Cuisines
- Approximate Cost for Two People
- Online Order Availability
- Table Booking Availability

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📊 Project Workflow
1. Import required libraries.
2. Load the dataset.
3. Explore the dataset using `head()` and `info()`.
4. Identify restaurant chains based on repeated restaurant names.
5. Filter restaurant chains.
6. Calculate the average rating for each chain.
7. Calculate the average number of votes for each chain.
8. Visualize:
   - Average Rating by Restaurant Chain
   - Popularity (Average Votes) by Restaurant Chain

## 📈 Results
- **Truffles** has the highest average rating (4.65).
- **Truffles** is also the most popular restaurant chain with an average of 1200 votes.
- **Meghana Foods** and **Empire Restaurant** also have high ratings and popularity.
- **Cafe Coffee Day** has the lowest average votes among the identified chains.

## ✅ Conclusion
The analysis shows that restaurant chains with higher customer ratings generally receive more customer votes, indicating greater popularity and customer satisfaction.
